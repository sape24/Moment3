Beskrivning av arbetsprocess

Började att skapa källkodsfilerna jag skulle arbeta med så som html, css och js. Sedan skapade jag ett repository av projektet på github och dubbelkolla att allting fungerade genom en commit. När jag var säker att det fungerade la jag till Parcel i projektet, med facit i hand så borde jag ha startat med parcel men det löste sig tillslut ändå node_modules hamna såklart i en .gitignore.


När parcel väl var tillagt och grundkonfiguration var gjord testade jag npm start samt npm run build för att se att allting fungerade. När jag visste att parcel fungerade började jag att fylla i html filerna med innehåll enligt uppgiften, när det kom till bilderna så använde jag mig utav parcels automatiserade optimering av bilder med hjälp av querystrings och picturetaggen. Detta för att på ett smidigt sätt kunna optimera originalbildern till ett modernt bildformat samt att webbläsaren väljer den bäst lämpade bilden att visa.


Jag dubbbelkollade att publiceringsfilerna i distkatalogen minifiera samt transpilera html, css och javascript koden och publicerade sedan webbplatsen på netlify.
